Steps

  1. In the primary navigation, open the Media module.
    2. >
  2. In the side navigation panel, click the Create folder link.
  3. Enter a name for the folder and then press enter.
  4. To add videos to a folder, select one or more videos.
  5. Click More ˇ > Move to Folder.
  6. Click on a folder and then click Move.

All folders will appear in the left navigation. The number after the folder name is the number of videos in the folder.

To remove videos from a folder, click on a folder, select one or more videos and then click Remove Videos ˇ > Remove Videos from Folder.

Deleting and renaming folders

Folders can be renamed or deleted by following these steps:

  1. Click the folder to be renamed or deleted.
  2. Click the down arrow next to the folder and click Rename or Delete.
