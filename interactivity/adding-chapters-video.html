Introduction

Chapters make it easier for viewers to go to a specific point in a video. This is especially helpful for longer form videos. Rather than scrubbing through the video, the chapter menu can be used to jump to a specific point in the video.

Note: Chapters can also be added to a video using a WebVTT file. For information see Adding Chapters to a Videos.

Working Sample

Adding a Chapter Menu

Note: Before you can add HapYak interactivity to a player, you need a HapYak-enabled Brightcove Player and you need to make sure a HapYak project has been created. For information on how to do this, see Getting Started with Interactivity.

To add a Chapter Menu, follow these steps:

  1. Log in to HapYak Studio.
  2. In the left navigation, click Manage.
  3. A list of projects will appear. Click your video in the project list.
  4. In the bottom toolbar, click the Chapter Menu.
  5. You can change the title of the Chapter Menu to anything or simply delete the text entirely to show a chapter menu icon.
  6. Chapters have several edit options to use:
    1. Start time - Is the point in the video at which the Chapter Menu will first appear.
    2. Duration(10s) - Is the total time the chapter menu will stay expanded. The default is 10s.
    3. Display Rule - Is an advanced option for more details, click here.
    4. Add Rule - Is an advanced option for more details, click here.
    5. Styles- Select a Styles option for the chapter menu. There are two options:
      • One column: It shows one column with the list of the Chapters.
      • Adaptive: It fills the screen with the list of Chapters.
    6. Initial State- It is set if you want the Chapter Menu to start expanded to collapsed when it first appears.
    7. Position- You can select the cardinal corner of the Chapters menu.
      8. Note: Brightcove recommends choosing either top-left or top-right options to navigate easily into the control bar.
  7. Click Done.

Adding Chapters

To add a new chapters, follow these steps:

  1. Play or scrub through the video. When you get to the point where a chapter needs to be added, click Add a chapter marker.
  2. Enter a short descriptive name for the chapter. The name will be shown to the viewer in the chapter menu.
    3. Note: The chapter label will only be displayed in the chapter menu, it will not be displayed during video playback.
  3. Continue playing the video and add a chapter at each major topic or section of the video.
  4. Confirm that all chapters appear in the Chapter Menu.
  5. Publish the video using the Media module and confirm that when the video plays, the chapter menu is visible.
