{{ page.title }}Adding Interactivity Chapters to a Video

{{ page.description }}

In this topic you will learn how to add chapters to a video using HapYak.

Introduction

Chapters make it easier for viewers to go to a specific point in a video. This is especially helpful for longer form videos. Rather than scrubbing through the video, the chapter menu can be used to jump to a specific point in the video.

Note: Chapters can also be added to a video using a WebVTT file. For information see Adding Chapters to a Videos.

Working Sample

Adding a Chapter Menu

Note: Before you can add HapYak interactivity to a player, you need a HapYak-enabled Brightcove Player and you need to make sure a HapYak project has been created. For information on how to do this, see Getting Started with Interactivity.

To use the Chapter Menu, follow these steps:

  1. In the bottom toolbar, click on Chapter Menu.
  2. Click Chapters Menu Icon, notice that immediately displays the Chapter Menu. You can change the name of the Chapter or delete the entire text.
  3. Click Edit.
  4. Chapters have several edit options to use:
    1. Start Time- Brightcove users can adjust the start time of the Chapter. Setting a start time is helpful for introductions of videos. You can delay the start time, and the Chapter will not appear until the specified time indicates it.
    2. 10s-Duration indicates how long the chapter menu will stand up without any direction using action.
    3. Display Rule
    4. Add Class
    5. Styles- You have two building styles options:
      • One column: It shows one column with the list of the Chapters.
      • Adaptive: It fills the screen with the list of Chapters.
    6. Initial State- You have two options:
      • Expanded.
      • Collapsed.
    7. Position- You can select the cardinal corner of the Chapters menu.
      8. Note: Brightcove strongly advises users to choose only top-left or top-right options to navigate easily into the control bar.
  5. Click Save.

Adding Chapters

To add a new Chapters, follow these steps:

  1. Click Add a Chapter maker in the part of the video where you want to create a new Chapter.
  2. It instantly shows a new Chapter. You can change the name.
    3. Note: The Chapter indicator label is a label that only is available in the editor. It will not be visible at the end published video.
  3. All Chapters appear in the Chapter Menu.

Publishing Videos

To publish videos with Chapters from Brightcove, follow these steps:

  1. Click Settings.
  2. In Integrations, check the published box.
  3. Log in to your Brightcove account > open the Media Module > Click the Video > Click the hapyak Player > Refresh the page.
