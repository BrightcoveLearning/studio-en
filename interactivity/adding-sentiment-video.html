HapYak Studio provides the ability to add different types of interactive overlays to videos. The User Sentiment widget can be used to gather viewer feedback on the helpfulness of videos by letting viewers rate their experience. Reports are available to provide the aggregate feedback about the ratings that were submitted.

Widget styling options

The User Sentiment widget offers three different styling options:

  • Thumbs Up/Thumbs Down
  • Star Rating (out of 5)
  • Emoticons

Notes on styling:

  • The widget can be positioned anywhere on the video
  • The widget cannot be resized
  • When the icon is clicked, it stays in the clicked/selected state for the remainder of the video

Adding a user sentiment widget

Typically, the User Sentiment widget is added to a template so that it can be easily added to multiple videos easily. Follow these steps to add a user sentiment widget to a template:

  1. Login to HapYak Studio.
  2. Click NEW TEMPLATE.
  3. Enter a Title and click NEXT.
  4. Confirm that the template is created. A sample video is automatically loaded into the template. To change the video, click Configure and enter a new Video Source.
  5. Click the widget icon in the bottom toolbar.
  6. Click the User Sentiment widget.
    Note: Your group must be provisioned to have access to the User Sentiment widget.
  7. Click ADD. The widget is added on top of a sample video.
  8. Hover over the widget and click CONFIGURE.
  9. Enter a USER PROMPT. This is the text that will appear at the top of the widget.
  10. Select a TYPE of widget.
  11. Choose to display the COUNT SETTINGS (displayed only for Thumbs type). A threshold can be set for each count which will prevent the count from displaying until a certain number of likes/dislikes have been recorded.
  12. Set the icon, font and color settings as needed.
  13. The DISPLAY SETTINGS control how the widget appears during video playback.
    • Beginning - Controls how the widget will appear when the video starts playing
    • Middle - Controls how the widget will appear as the video is playing
    • End - Controls how the widget will appear at the end of the video

    The following example would display the sentiment widget for 5 seconds when the video starts playing, then it would minimize and then expand for the last 5 seconds of the video.

  14. Click SAVE.

A tag can be added to templates as a way to quickly add the User Sentiment widget to Video Cloud videos that are published using a HapYak-enabled player. If a published video has the same tag that was assigned to the template, the User Sentiment widget will be displayed during video playback.

Follow these steps to add a tag to the template.

  1. At the top of the page, click Configure.
  2. Locate the Tags field and click it.
  3. Add one or more Tags. This example is using a tag of sentiment.
  4. Click SAVE and then SUBMIT.

By adding the tag sentiment to the template, any Video Cloud video that is tagged sentiment and published using a HapYak-enabled player will have the User Sentiment widget displayed during video playback.

Note: There may be a delay of up to one hour before the User Sentiment widget will appear in published Video Cloud videos.

Reviewing sentiment reports

The User Sentiment widget will collect a "rating" when the viewer clicks on their satisfaction emoji. Since there are three types of emojis which have been implemented (thumbs, stars and a smilies), each has its own rating map which is then normalized into a score from 1-5.

The User Sentiment Summary report provides information on all projects with the User Sentiment widget. To access the report, follow these steps:

  1. Login to HapYak Studio.
  2. In the left navigation, click Core Reports.
  3. Select a Date Range for the report.
  4. At the top of the page use the report selector to select the User Sentiment Summary report.
  5. Click RUN.

The report will display all projects with the User Sentiment widget.
