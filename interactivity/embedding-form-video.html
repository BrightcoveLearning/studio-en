The Custom Form widget allows you to embed any form from any system such as Google Forms, Typeform, Calendly, Eventbright and others on top of a video. The Custom Form widget can also be used to display external web pages.

This widget also cleverly hides the form and only slides it open when a viewer clicks a call to action.

Using an external form (like a Google Form) with this annotation has the advantage of sending all the submitted data straight to that system. Note that no data submitted to the form will show up in the standard HapYak reports as it's routed directly to the external system.

Think about setting the Start Time to 5 or 10 seconds (near the beginning) and setting the duration to "Until the end of the video". This will allow the viewer to click the CTA at any point in the video while keeping the form out of sight until the CTA is clicked.

Before you get started

Before you can add HapYak interactivity to a player, you need a HapYak-enabled Brightcove Player and you need to make sure a HapYak project has been created. For information on how to do this, see Getting Started with Interactivity.

Adding a custom form widget

Using an external form (like a Google Form) with this annotation has the advantage of sending all the submitted data straight to that system. Note that no data submitted to the form will show up in the standard HapYak reports as it's routed directly to the external system.

Follow these steps to add a Custom Form (this topic uses a Google Form) widget to a project:

  1. Login to HapYak Studio.
  2. In the left navigation, click Manage.
  3. Click on a project to open it.
  4. In the bottom toolbar, click on the widget icon.
  5. Click the Custom Form widget.
  6. Click ADD. The widget is added on top of the project video.
  7. Click and drag the widget to where you would like the text to appear over the video.
  8. Click on the Done link in the menu.
  9. Hover over the Your Text Here text and click CONFIGURE.
  10. In the FORM SOURCE field, paste in the form URL.
  11. Update the Button Text, this is the text that the viewer will click to open the form.
  12. Click SAVE.
  13. Click Edit to open the editor.
  14. Click the Start Time menu.
  15. Enter a Start time in seconds, as a percentage or in time format.
  16. Click Submit.

A tag can be added to templates as a way to quickly add the User Sentiment widget to Video Cloud videos that are published using a HapYak-enabled player. If a published video has the same tag that was assigned to the template, the User Sentiment widget will be displayed during video playback.

 

Note: There may be a delay of up to one hour before the Custom Form widget will appear in published Video Cloud videos.

Displaying a webpage

The User Sentiment widget will collect a "rating" when the viewer clicks on their satisfaction emoji. Since there are three types of emojis which have been implemented (thumbs, stars and a smilies), each has its own rating map which is then normalized into a score from 1-5.

The User Sentiment Summary report provides information on all projects with the User Sentiment widget. To access the report, follow these steps:

  1. Login to HapYak Studio.

The report will display all projects with the User Sentiment widget.
