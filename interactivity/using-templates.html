A template is a track of annotations that is applied to multiple HapYak projects. Templates can be used to quickly apply identical annotations to multiple videos with one click.

Multiple template tracks can be added to any project. If the template track is updated, HapYak will automatically update all projects using this template. This is great for any interactive features that should be added to a large number of videos. Some typical use cases are:

  • Adding a logo to all of the company videos
  • Adding a "skip-ahead" button to a series of videos that have the same pattern
  • Adding a call-to-action on all videos with a similar subject

Creating a template

To create a template, follow these steps:

  1. Login to HapYak Studio.
  2. Click NEW TEMPLATE.
  3. Enter a Title and click NEXT.
  4. Confirm that the template is created. A sample video is automatically loaded into the template. To change the video, click Configure and enter a new Video Source.
  5. Use the toolbar below the video to add annotations.
    Tip: If the annotation Start Time is set to -20, the annotation will appear 20 seconds before the end of the video. This is a good way to have the annotation dynamically adjust for every project the template is applied to. Likewise, setting the Start Time to 25% will set the start time to the 25% point of the video, no matter the length.
  6. To apply the template to a project, click on a project to open it.
  7. Click SETTINGS at the top of the page.
  8. Locate the Templates Applied field and click it.
  9. Click on one or more templates to add to the project. Templates added to the project will be listed at the top of the dialog.
  10. Click SUBMIT.
