Overview

The Campaign 2.0 migration involves updating existing video marketing campaigns to leverage the enhanced capabilities of the latest Brightcove features. These include a new Player plugin for enriched video experiences, updated client-side data capture for marketing tools such as Marketo and Hubspot, and new interactivity options through Lead Forms integrated directly within the Brightcove Player.

Note: The features and integrations discussed in this document apply only to Video On Demand (VOD) content. They are not available for live-streaming events within the Brightcove platform.

Supported third-party platforms

The following third-party platforms are currently supported for this workflow:

  • Marketo
  • Hubspot
Note: There are multiple methods to collect viewer ID and viewer's data within the Brightcove ecosystem. For instance, the setUser function or set the Player Plugin. In this documentation, we will concentrate on the Player Plugin in conjunction with Interactive Lead forms.

Set up an integration

Setting up an integration is straightforward. However, you will need some information from the third-party vendor - exactly what you need will vary for different services, but will be clearly indicated by input fields in the setup dialog. Typical requirements are "Account id", "API key", and/or "Secret key". If you are unsure how to find or obtain a required piece of information, consult the vendor's documentation or support service.

Steps

  1. Login to Studio
  2. Click the Integrations icon in the left-side navigation to open the Integrations module:
  3. Click Customer Data Platforms to go to the data platform integrations:
  4. As you see in the screenshot above, any existing integrations will be shown as a list and can be modified (see the next section). Here we will click Add Integration to create a new integration.
  5. In the Add Integration dialog, first select the service you want to integrate:
  6. Next, fill in the required information according to the service you are integrating:
  7. Click Save to complete the setup.

Modify an integration

As shown above, in the list of existing integrations, there are three icons that allow you to:

  • Edit the integration
  • Test the integration
  • Delete the integration

In some cases the Test icon will be grayed out, indicating that this functionality is not available for that particular integration.

If you click the Edit icon, a dialog similar to the Add Integration dialog will appear:

Note the additional Integrations Properties tab. For some services, there will be additional properties you can set here; for others there are no additional properties.

Follow these steps to disable your Campaign Enabled players:

  1. Login to Video Cloud Studio.
  2. Open the Players module.
  3. Click the link for the Player you want to open the player properties page.
  4. Click Campaign in the left navigation menu.
  5. Disable the Enable Campaign checkbox.
  6. Save
  7. Publish

Enable the Viewer ID Tracking

Follow these steps to enable the Viewer ID Tracking plugin from the player details view:

  1. Click the Viewer ID Tracking in the left navigation menu.
    1. Click the Enable Viewer ID Tracking checkbox.
    2. Choose the 3rd Party Service from the Service dropdown (Hubspot, Marketo).
    3. Provide your respective Account ID.
  3. Save
  4. Publish

Set up MAPs Tracking

To Setup MAPs (Marketo & Hubspot Client-Side Only) from the player details view, follow these steps:

  1. Click the MAPs in the left navigation menu.
    1. On the MAPs page, select the Enable MAPs checkbox.
    2. Select your Service.
    3. Provide your respective Account ID.
  3. Save
  4. Publish

Interactivity Player Set Up

To Set Up your Player to automatically create Interactive Projects that can receive Lead Forms, from the player details view, follow these steps:

  1. Click Interactivity in the left navigation menu.
    1. Check Enable Interactivity.
    2. Check Options (JSON) to enable Interactivity projects auto-creation. This reveals a textarea where you can insert a JSON configuration.
    3. Inside the textarea, insert the following JSON. 
      {
   "create": true
  }
  3. Save
  4. Publish

Note: Once the changes have taken effect, any video played within this Player will have a Project created automatically.

When creating Projects from videos, any existing tags in the Media Module are copied to the Project. To better manage videos with lead forms, consider assigning them a unique tag in the Media Module.
