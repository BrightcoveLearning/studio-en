The Control Room page is used to view/edit information about the live event.

Begin Streaming

To connect the video stream to your encoder:

  1. Open your encoder to start streaming.
  2. Click Copy in the Streaming Endpoint (RTMP URL) and paste it into your encoder.
  3. Click Copy in the Stream Name and paste it into your encoder.
  4. With that information, start the streaming on your encoder, and the video will appear in the Control Room in a few seconds.
  5. To stop the event, click on Cancel Stream.

Live event information

The Control Room page displays five main sections:

  • Event Information - Displays information about the event as Event Status, Current Viewers, and Elapsed Time.
  • Event Details - Displays reference details of the event as Event Name, Video ID, and Short Description.
  • Event Configuration Settings - Displays information about the configuration settings of the event as Job ID, Video ID, Status and Endpoint.
  • Monitor - Retrieves data and graphs from the current event.
  • Renditions - Displays details about the renditions that were created. Clicking on a rendition link will show the encoding settings for the rendition.
  • Clips - Displays all of the clips that were created from the live stream.
