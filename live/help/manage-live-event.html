The Manage Events page allows you to manage your Live, Standby, and Completed events as well as creating a new one.

Create Event

Events are created to stream shorter length, one time or recurring live events.

  1. To create a new event, click Create Event.
  • Learn more
Create Channel

Channels are used when delivering 24/7 live events.

  1. To create a new channel, click Create Channel.
  • Learn more
Manage live events

The Manage Events page displays three main tabs:

  • Live - Lists live events that are in progress.
  • Standby - Lists recurring events that are in standby mode.
  • Completed - Lists completed events.
