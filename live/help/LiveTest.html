Live Home

The Live module is used to broadcast live events to both desktop and mobile devices.

Create a Live Event

Click Create Event, and fill in the following information:

  • Event Name
  • Short Description
  • At the bottom, click the Create Event button to start the live event

Control Room
The Control Room contains all the information and settings of your event, in this window:
  • Copy the Job Id and the Video Id from the Test Event panel.

You will need both Id's to configure the Live event in the Brightcove Beacon Event settings.

Connect your Encoder
To start streaming in the Live platform, you will need an Encoder, for information on how to use an encoder please select:
  • Step-by-Step: Using the Live Module with Telestream Wirecast
  • Step-by-Step: Using the Live Module with Open Broadcaster Software (OBS)
