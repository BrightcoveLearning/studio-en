Brightcove reserves the right to update the default ingest profiles associated with Video Cloud accounts. You may also have the need to update your own, custom ingest profiles. When ingest profiles change, you may wish to have existing videos in your account retranscoded so that they have renditions that correspond to the new ingest profile settings.

Note: The ability to retranscode existing videos is available only for Professional and Enterprise accounts. If you are interested in this feature, please contact your Account Manager.

Videos can be retranscoded using the Media module or by using the Dynamic Ingest API. Using the Dynamic Ingest API may be a better solution when you have a large number of videos that require retranscoding. To retranscode videos using the Dynamic Ingest API, see the Dynamic Ingest API Overview developer document.

Note: You cannot retranscode videos that have been shared to your account.
Note: Only videos that were uploaded using the Upload module or using the Dynamic Ingest API, can be retranscoded. If the video can't be retranscoded, the retranscode will fail.
Note: videos can only be retranscoded if the video master was archived when the video was added to the account. Older videos may not have a master, depending on the ingest profile that was used.

Steps to retranscode a video

Follow these steps to replace the existing renditions for a video.

  1. Open the Media module.
  2. Click on the video you want to retranscode.
  3. In the Video Files section, click Retranscode.
  4. Select an ingest profile to use.
  5. Check Keep existing thumbnail and poster images to keep the existing images. If you uncheck this box, new images will be generated based upon the image settings in the selected ingest profile.
    Note that if you check the box, the existing images will be preserved - however, the image URLs will change.
    Note: If the account is enabled for forensic watermarking, there will be an option to either add a forensic watermark or keep the existing forensic watermark.
  6. Click Retranscode.

While the new renditions are being created, the video will remain playable using the existing renditions.
