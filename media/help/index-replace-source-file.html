Upload - Replace Source

{{ page.description }}

  1. Select an Ingest Profile. The Ingest Profile will determine the number and quality of renditions that are created.
  2. To select a new video source:
    • Drag and Drop files from the file system to the Upload module OR
    • Click Browse for Videos and browse and select one or more files to upload OR
    • Click Enter URL and enter the URL to a remote video file
  3. The file will begin processing. Remain on the Upload module page (and don't refresh the page) until the video has a status of Processing Complete.
