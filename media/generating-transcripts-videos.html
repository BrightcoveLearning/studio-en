Generating Transcripts for Videos

{{ page.description }}

The Upload and Media modules both provide the ability to automatically generate video transcripts. The generated transcript is saved as a WEBVTT file and can be used to display video captions.

Generating transcripts when uploading

The Upload module provides the option to generate a transcript file when a video is uploaded. To generate a transcript file during upload, follow these steps.

  1. Click Upload in the navigation header to open the Upload module.
  2. Enable the Auto transcribing option, the Automatic transcribing settings dialog will open.
  3. Select the Language spoken in the video. If a Language is not selected, the video language is auto-detected. Selecting a language will improve the transcript generation time and may improve overall accuracy.
  4. Select a Label. The label is typically set to the language used in the video. The label will be displayed in the CC dropdown in the video player and will be used by viewers to select the captions to be displayed.
  5. Click Done.

Downloading transcript files

Chances are the transcript generation process will not be perfect and you may need to edit things like names used in the video or punctuation. The Media module provides the ability to download a transcript file. To download a transcript file, follow these steps.

  1. Click Media in the navigation header to open the Media module.
  2. Click the link for the video you want to edit.
  3. Locate the Text Tracks section and click Edit.
  4. Click the download icon () in the Actions column to download the transcript.

The transcript file can be edited using any text editor. Once the changes are made, delete the existing text track file by clicking on the delete icon () in the Actions column. The edited file can then be uploaded. For information on how to upload a new text tracks file, see Configuring the text track properties.
