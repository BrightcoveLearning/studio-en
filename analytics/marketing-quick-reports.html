Overview

Quick Reports offers a streamlined way to generate and access predefined reports for key data insights. By selecting specific report types and date ranges, you can quickly create reports on content performance, audience behavior, and viewer-level data. Quick Reports allows you to efficiently manage and analyze key video performance metrics.

Navigation

  1. Log in to your Video Cloud account.
  2. Click on Analytics
  3. On the left navigation menu, Click Quick Reports

Create Report

To create a quick report, follow these steps:

  1. Choose the report type you want to generate from the drop-down menu.
  2. Set the Start Date and End Date using the date picker.
  3. Create Report

Learn more about the report format: Viewing Data Integration Properties

Note: Quick Reports analytics uses a separate system from the general Video Cloud analytics system, and the two-track data are different. Therefore, you will see differences in some metrics if you compare the two. For example, Video Cloud analytics counts views as soon as a video starts. However, Quick Reports analytics only counts 'engaged' views. A view is 'engaged' if the watch duration is at least 10% of the video duration or 30 seconds (whichever is smaller). So for a 1-minute video, if 14 viewers start the video, but 6 of them stop it before 6 seconds have played, Video Cloud analytics would show 14 views, while Quick Reports would show 8. For videos longer than 5 minutes, the cutoff is 30 seconds.

Dashboard

The Quick Reports dashboard offers an organized view of the essential details for the quick reports generated.

Date Range: Indicates the start and end dates for the report.

Status: Displays the current status of the report.

Date Created: Shows the date the report was generated.

Expires: Shows the report's expiration date.

Actions: Provides options to download or delete the report.

Troubleshooting

Missing or Partial Data

Data is typically processed within 48-72 hours and refreshed around 15:00 UTC. Running a report before this may result in missing or partial data for the last 48-72 hours. If you see missing data past 72 hours, please contact Brightcove Customer Support.

Running a report multiple times does not refresh data

Quick reports are cached for 3 hours. Running multiple reports with the same date range within this period will return the same report.

Live videos are not included in report

Currently, live videos are not included in the report because Brightcove cannot calculate accurate watch percentages for these events.
