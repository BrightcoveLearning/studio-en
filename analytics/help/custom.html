Custom Reports

Create custom analytics reports that show selected data and metrics for a period of time.

Report time zone

Data in custom reports is based on the account's reporting time zone; to configure your time zone:

  1. In Beacon Studio, expand the Admin dropdown.
  2. Select Account Settings.
  3. Set your Current Time Zone.
  4. Click Save.
Changing the reporting time zone only affects data going forward, and changes are not applied retroactively.

Report data
  • Include Accounts - Include data from the current Video Cloud account or select multiple accounts
  • Report Type
    • One-time - Report is run once, immediately
    • Date Range - For one-time reports, specify the time period for analytics data
    • Recurring - Report can be scheduled to run on a daily, weekly or monthly basis
    • Frequency - For recurring reports
      • Daily - Report covers an entire day
      • Weekly - Report covers an entire week, defined as Monday through Sunday
      • Monthly - Reports covers an entire month
      • Recurring reports are executed after analytics reconciliation, usually 3 to 5 days after the end of a reporting period.
  • Data Groups - Data Groups determine the data fields included in the report
    • Data Fields that appear are based upon the data groups you select
    • You can select multiple data groups but some combinations are not valid
    • At least one Data Group must be selected
  • Data Fields - When you select a Data Group, the Data Fields that can be returned are selectable
    • Grayed-out fields cannot be returned for the selected Data Group
  • Click Save to save your changes

Add report filters

Filters can be added to limit the amount of data returned.

  • When using multiple filters, an AND operation will be used
  • A filter will only display in your report if you have selected its matching data group
    • For example, if you want to filter on Video Tag, you must select the Video dimension

You can filter on the following:

  • Video - One or more active video IDs in the account
  • Video Tag - One or more video tags
  • Custom Fields - One or more custom fields in the account
  • Player - Enter one or more player IDs, separated by a comma
  • Source Type
  • Device Type
  • Device OS
  • Device Manufacturer
  • Browser Type
  • Search Terms
  • Country
  • Destination Domain
  • Search Terms

Preview Report

The data displayed in the report preview is sample data, not actual data from your Video Cloud account.

  • The Preview Report section shows the data columns selected, populated with sample data
  • Click to delete the associated data item
  • The sort order of the preview has no effect on the sort order of the downloaded report
  • You can use a spreadsheet application to sort the downloaded data

Configure and Deliver Report
  • Report Name - Start by giving the report a meaningful name
  • Format
    • CSV - Comma-separated values
    • XLSX - Microsoft Excel
  • Email me when my report is ready - Get an email notification to yourself when the report is complete
  • Notify others - Enter email addresses separated by a comma
  • In the Brighcove email, the link to download the report is valid for 90 days
  • For one-time reports,
    • Click Request Report
    • The report will be downloaded when it is complete
  • For recurring reports,
    • Click Schedule Report
    • The report will be queued and run when the data is ready
  • To see all scheduled reports, click Scheduled Reports in the left navigation
