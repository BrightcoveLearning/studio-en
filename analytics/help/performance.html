Performance

Get fast, close to real-time analytics data for your videos and players.

Performance data

The performance report shows metrics, including video views, traffic sources, device types, device operating systems, and more.

  • Video Views dropdown - Select the report metrics:
    • Video Views - Number of times a video starts playback
    • Video Impressions - Number of times the video was loaded into a player and ready for interaction
    • Play Rate - Video Views divided by Video Impressions
    • Viewed Minutes - Total number of minutes the video was viewed
    • Engagement Score - Average percent watched per video view
    • Daily Unique Viewers - Average daily unique viewers for the time period selected
    • Video Bandwidth - Bandwidth for videos only; may be delayed by 3-5 days
    • Total Bandwidth - Includes bandwidth for videos, images and player loads; may be delayed by 3-5 days
  • Performance dropdown - Switch between the Performance and Engagement reports
  • Device dropdown
    The graph shows the number of video views per day for the time period selected.

    The report body shows the following, when each dimension is selected:
    • Video - Video name, video ID, duration and performance details for each video
    • Player - Performance details for each player
    • Source of Traffic - Performance data for each traffic source
      • Referral - User followed a link to the video
      • Tablet - Includes iPads, Android tablets, etc.
      • Mobile - Mobile phones
      • Connected TV/Game Console/Companion Device - Includes Smart TVs, set-top boxes and game consoles
      • Other - User-agents not mapped to any of the above
    • Display Domain - Performance data for the domain the user was on when they watched a video
    • Country - Performance data for each country where videos were viewed
    • Device - Performance data for each Device Type/Device Operating System combination that was used to view a video
    • Device Type - Performance details by device type
      • Desktop - Includes desktop and laptop computers
      • Direct - User typed the link directly or the referrer was not determined
      • Secure search - User logged in with a Google account and performed a search
      • Organic search - User performed a search and clicked on a result
      • Paid search - User performed a search and clicked on a paid result
    • Device OS - Performance data for each device operating system
    • Device Manufacturer - Performance data for each device manufacturer
    • Browser Type - Performance data for each browser type
    • Day - Performance data for each day

Add filter

Use filters to limit the data that appears in your report.

  1. Click the + Add filters pill.
  2. Analytics data can be filtered by tags, custom fields, countries and device types.
    Configure the filter criteria using the filter panel.
  3. Click Apply Filters.

Export report data

Report data can be downloaded as either a CSV file or Excel formats.

The export saves all the data directly from an Analytics API call, without filtering. This allows you to manipulate the data as you want.
  1. To the right of Export as, click either CSV or XLSX.
  2. Open or save the file to your computer.
