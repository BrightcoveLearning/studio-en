Scheduled Reports

Manage your custom analytics reports that you have set to run on a recurring basis.

Enterprise accounts can have up to 20 scheduled reports. Professional accounts are limited to 5 reports.
Report list
  • Name - Scheduled report name
  • Frequency - Time interval that the report will be run
  • Status of Last Report - Whether the last report was delivered or not
  • Report Recipients - List of email addresses that will receive the report
  • Active or Inactive - Set the status for the report
    • Active - The report is scheduled to be processed
    • Inactive - The report will not be processed
Report details
  1. To expand report details, click + next to the report name.
  • Date created - Date that the report was run
  • Accounts included - List of accounts from which the data was collected
  • Data groups - Data groups included
  • Filters - Filters used to limit the results
  • Group data - Group and data fields included
  • Performance data - Video and player performance metrics included
  • Engagement data - Data that shows how users are engaging with your videos
  • Utilization data -
  • Custom fields - Custom fields included
  • Sorted by - Data columns sorted
  • Email notifications to - List of email addresses that receive the report
  • Next scheduled report - Date the report will be run next
Edit report
  1. To expand report details, click + next to the report name.
  2. Click Edit.
  3. Make changes to the report definition.
  4. Click Schedule Report.
If the report includes multiple Video Cloud accounts, you must have access to all accounts to edit the report.
Run report
  1. To expand report details, click + next to the report name.
  2. Click Run report.
Data currently available for the next scheduled report will be downloaded.
Duplicate report
  1. To expand report details, click + next to the report name.
  2. Click Duplicate.
  3. Make changes to the report definition.
  4. Update the Report Name.
  5. Click Schedule Report.
The report type can be changed from Recurring to One-time, allowing you to run the report immediately.
Delete report
  1. To expand report details, click + next to the report name.
  2. Click Delete.
  3. To confirm deletion, click Delete Report.
If the report includes multiple Video Cloud accounts, you must have access to all accounts to delete the report.
