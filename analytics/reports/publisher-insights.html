Introduction

Publisher insights is aimed to provide relevant and time-sensitive insights about your viewer's consumption habits, which will help you make informed decisions about your content schedule.

This module consists of three tabs, including Trending, Engagement, and Content.

Data is regularly updated, ranging from minute-by-minute to a day's delay.
If you would like publisher insights enabled for your account, please contact your customer success manager.

Set up

Publisher insights is available to Media Studio Premium customers. Contact your Customer Success Manager to enable your account.

Navigation

  1. Log in to your Video Cloud account.
  2. Click on Analytics
  3. On the left navigation menu, Click Insights

Trending

The trending tab contains a dynamic snapshot of your platform's most current and engaging content. This feature brings together key metrics to help you understand and capitalize on the latest trends, ensuring your audience remains captivated.

You can find more information about this tab here.

Engagement

The engagement data is crucial for comprehending subscribers' usage patterns and establishing a baseline success rate to encourage customer retention. It provides insights into audience interaction with the library, including detailed trends in video consumption, average days streamed, and activity over time.

You can find more information about this tab here.

Content

The content data is essential for identifying content performance and library effectiveness. It enables a detailed analysis of audience interaction, featuring a content explorer and a library breakdown for popular series.

You can find more information about this tab here.

Configuring the Content Owner video metadata field

  1. Log in to Video Cloud Studio.
  2. Open the Admin module and then click Video fields.

    Note: You must be an account administrator to access the Admin module.
  3. The Video Fields page will display your custom and standard video metadata fields.
  4. Click Add Custom Field.
  5. Enter Insights Content Owner as the Display Name.
  6. Enter insights_content_owner as the Internal Name.
  7. (Optional): Enter a Description. The description field will appear as a hint in the Studio interface.

    Note: The internal name automatically changes when you change the display name. The internal must always be insights_content_owner.
  8. Select a Type for the field:
    • Text: The user can type anything into the field up to 1024 single-byte characters (or 512 double-byte characters).
    • List: The user will select from a list of values. The list can have no more than 150 possible values, and each value can be 100 characters long.
  9. If your field type is List, enter a comma-separated list of values and click Add to List.

    Requirement: You must include the value -MY_CONTENT- if the field is required.
  10. If the custom field will be required, check Make this Field Required.
    Note: Content will be set to Inactive if Required Fields are not filled in.
  11. Click Save Field to save the changes.

Updating the Content Owner video metadata field

  1. Log in to Video Cloud Studio.
  2. Open the Media module.
  3. Click on the video you want to update.
  4. Go to the Custom Fields section located at the bottom.
  5. Click Edit

  6. Go to Insights Content Owner custom field.

  7. Type or select the value.

    If the field is required you must type or select: -MY_CONTENT- to denote your content.
    8. See the steps outlined in “Configuring the Content Owner video metadata field” if you do not see the value.
