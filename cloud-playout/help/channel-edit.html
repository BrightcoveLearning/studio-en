Add Videos
  1. Click Add Videos.
  2. Select the videos you want (you can also search for videos).
  3. Click Add Videos.
  4. Click Save to save the changes to the channel.
  • Learn more
Import Playlist
  1. Click Import Playlist.
  2. Select the playlist you want and click the Import button beside it.
  3. Click Save to save the changes to the channel.
      • Learn more
Add Static Overlay
  1. Click Overlays.
  2. Click Static:
  3. Click Add New.
  4. Select a layer (only matters if you're adding two overlays that will overlap - 1 means on top).
  5. For the overlay image, you have already defined one, you can select it from the dropdown, or otherwise select Add New Image.
  6. If this is a new overlay, enter a public URL where the image resides in the Image URL field.
  • Learn more
Add Dynamic Overlay
  1. Click Overlays.
  2. Click Static:
  3. Click Add New.
  4. Select a layer (only matters if you're adding two overlays that will overlap - 1 means on top).
  5. For the overlay image, you have already defined one, you can select it from the dropdown, or otherwise select Add New Image.
  6. If this is a new overlay, enter a public URL where the image resides in the Image URL field.
  • Learn more

Add Live Event

Note this applies only if the channel is configured to include live events.

  1. Click Add Live Event.
  2. Add a name and description, and set the duration of the event (you can increment by 5 seconds or type directly in the Duration field).
  3. Click Save.
  • Learn more
Quick Edit

Note this applies only to live events in the program.

  1. Select a live event.
  2. Click Quick Edit.
  3. Adjust the duration of the event.
  4. Click Save.
  • Learn more
Reorder Videos

Note this applies only if you have added videos and/or live events to the channel.


  1. Click the video asset you wish to move.
  2. Drag it to the new location and drop it.
  • Learn more
Delete Videos

To remove videos from the channel (this will not delete them from your account):

  1. Select one or more videos.
  2. Click Delete.
  3. Adjust the duration of the event.
  4. Click Save.
  • Learn more
Activate Channel
  1. Click Activate (note that because of a UI issue, the Activate button on the upper right corner is currently partly cut off).
  2. The channel will play automatically at the start date/time, but you can start it immediately by clicking the Play Now button.
