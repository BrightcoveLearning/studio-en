Show Playing Now

Edit Program
  1. Click Edit Program.
  2. Add, delete, or re-order videos. (Reorder by clicking and dragging the :: icon.)
  3. Click Save.

Changes will apply to future playing items.

  • Learn more
Add Static Overlay
  1. Click Overlays.
  2. Click Static:
  3. Click Add New.
  4. Select a layer (only matters if you're adding two overlays that will overlap - 1 means on top).
  5. For the overlay image, you have already defined one, you can select it from the dropdown, or otherwise select Add New Image.
  6. If this is a new overlay, enter a public URL where the image resides in the Image URL field.
  • Learn more
Add Dynamic Overlay
  1. Click Overlays.
  2. Click Static:
  3. Click Add New.
  4. Select a layer (only matters if you're adding two overlays that will overlap - 1 means on top).
  5. For the overlay image, you have already defined one, you can select it from the dropdown, or otherwise select Add New Image.
  6. If this is a new overlay, enter a public URL where the image resides in the Image URL field.
