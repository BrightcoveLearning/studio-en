Add new channel
  1. Click to open the Create Channel dialog.
  2. Enter a name (English only, no spaces; these special characters are allowed: _ . : / = + -).
  3. Enter a description of the channel.
  4. Click on the Start Date/Time field to select a start date and time.
  5. If you toggle on, you may also enter an optional End Date/Time
  6. If you do not wish to import any available 608 closed captions embedded in videos, toggle the option off. See Importing Captions
  7. Toggle on to enable SSAI - it will also remove the Live Input option. If you toggle this on, you can also toggle Import Mid-Roll Cue Points for mid-roll ads, and change the duration of mid-roll ads, if you wish.
  8. Toggle to include live events in your channel. You will also need to provide a URL for slate content or use the default slate to fill time when the live stream isn't available.
  9. Select Mobile & Web or Smart TV for the output.
  10. Select the Output Region closest to the majority of your audience.
  11. Select Brightcove Live and then Create New Brightcove Live Channel as the destination to output to.
  12. Click to create the channel.
