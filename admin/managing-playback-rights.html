Playback rights are used to restrict video playback. After playback rights have been created, they can be assigned to videos using the Media module.

To access the Playback Rights administration page, click the ADMIN link and then Playback Rights. A list of playback rights will appear.

Note: To access the Playback Rights page, you must be a Video Cloud Studio user with the Administrator role.
Playback rights are assigned to videos in the Playback Restrictions section of the video properties page in the Media module.

Playback rights with a Status of Inactive cannot be assigned to videos in the Media module.

Creating playback rights

To create playback rights, follow these steps.

  1. Log in to Video Cloud Studio as an Administrator.
  2. In the navigation header, click ADMIN and then Playback Rights. Note that you must be an account administrator to access to the ADMIN menu.
  3. Click New Playback Rights.
  4. Enter a Playback Rights Name.
  5. Checking Allow Playback Right without token will allow playback requests to be made without a JSON Web Token (JWT).
    Note: Unchecking this option won't affect the restrictions set for that Playback Right, however, the playback request is going to require a valid JWT which is set programmatically at the player level. For information, see the developer document Using Playback Authorization DRM with the Brightcove Player.
  6. Check Use as default to make this the default playback right. This will apply the configured playback right restrictions to uploaded videos.
    Note: The default playback right is not assigned if a video is re-transcoded.
  7. Click Save.
Note: The maximum number of playback rights you can have in an account is 100. If you need to increase the limit, please contact Brightcove Support.

Click a restriction link in the left navigation to add restrictions to a playback right:

  • Geographic
  • Domain
  • Proxy
  • Time

Editing playback rights

To edit playback rights, follow these steps:

  1. Click on a playback right.
  2. Make required changes and click Save.

Duplicating playback rights

To duplicate a playback right, follow these steps:

  1. Click on a playback right.
  2. Click Duplicate.
  3. Make required changes and click Save.

Activating/Deactivating playback rights

Playback rights cannot be deleted, however, they can be deactivated so they can't be assigned to videos in the Media module. To activate/deactivate playback rights, follow these steps:

  1. Click on a playback right.
  2. Click Activate to activate a playback right or Deactivate to deactivate a playback right. Playback rights that are deactivated cannot be assigned to videos in the Media module.

Assigning geographic restrictions

Geographic restrictions allow you to limit video playback based upon country, ZIP code, IPs and designated market area. To assign geographic restrictions to a playback right, follow these steps:

  1. Click on a playback right.
  2. In the left navigation, click Geographic Restrictions.

The Global rule can be set to Allow all or Block all.

  • Allow all
    • Used when you want to blacklist a small number of groups who will not have access
    • Select to allow viewing for all and then add a list of blacklisted countries, zip codes, IPs and DMAs that are prevented from viewing
  • Block all
    • Used when you want to whitelist a small number of groups who will have access
    • Select to block viewing for all and then add a list of whitelisted countries, zip codes, IPs or DMAs that are allowed to view

Geographic restrictions can be added to allow/prevent viewing based upon:

  • Countries
  • ZIP Codes
  • IP Restrictions
  • Designated Market Area (DMA)

Restricting access to countries

Country playback rights can be used to limit viewing to specific countries.

  1. Select the countries that will be prevented (blacklisted) or allowed (whitelisted) to view. As you type, the selection list will be filtered.
  2. Click Save.

Restricting access to zip codes

Zip code playback rights can be used to limit viewing to specific zip codes.

  1. Enter a list of zip codes that will be prevented (blacklisted) or allowed (whitelisted) from viewing. The zip codes should be prefixed with an uppercase, two letter country code and hyphen, for example, US-90045.

It is also possible to upload a .csv file containing zip codes. Click Upload CSV to browse and select a file to upload.

Note: Uploading a .csv file will replace the contents of the ZIP codes field.

Restricting access to IPs

IP playback rights can be used to limit viewing to an IP address range.

  1. Enter a range of IP addresses that will be prevented (blacklisted) or allowed (whitelisted) from viewing. CIDR block notation is also supported.
  2. Click Save.

Restricting access to DMAs

A designated market area (DMA), also referred to as a media market, is a region of the United States that is used to define television and radio markets. Playback rights can be used to limit viewing based upon DMAs.

  1. Enter a list of DMAs that will be prevented (blacklisted) or allowed (whitelisted) from viewing
  2. Click Save.

Assigning domain restrictions

Domain restrictions allow you to limit video playback based upon domain. To assign domain restrictions to a playback right, follow these steps:

  1. Click on a playback right.
  2. In the left navigation, click Domain Restrictions.
  3. In the Blacklisted Domains field, enter the domains that will be prevented from viewing.
  4. In the Whitelisted Domains field, enter the domains that will be allowed to view.
  5. Click Save.

Assigning proxy restrictions

Proxy restrictions allow you to limit video playback based upon certain types of connections. To assign proxy restrictions to a playback right, follow these steps:

Note: As VPN providers continue to improve their services, resolving IP addresses may change from what you have defined for blocked IP addresses. Because of this, proxy blocking is not 100% guaranteed.
  1. Click on a playback right.
  2. In the left navigation, click Proxy Restrictions.
  3. Select the proxy types to block.
  4. Click Save.

Assigning time restrictions

Time restrictions allow you to limit video playback based upon a specified start date. To assign time restrictions to a playback right, follow these steps:

  1. Click on a playback right.
  2. In the left navigation, click Time Restrictions.
  3. Click the calendar control to select a Start Date.
  4. (Optional) Click Add End Date to select an end date.
  5. Click Save.
