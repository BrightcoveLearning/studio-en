When uploading videos using the Upload module or when creating a Dynamic Ingest request using the DI API, you must supply an ingest profile which determines the number and type of renditions created by the Video Cloud ingestion process. This topic provides detailed information on each of the legacy ingest profiles. For information on the Dynamic Delivery profiles, see Standard Ingest Profiles for Dynamic Delivery.

Note: When lower quality (lower bitrate) source content is uploaded and one of the standard profiles is selected, Video Cloud will only produce renditions that are equal to or lower in quality than the source.

It is also possible to create your own custom ingest profiles. For information, see Managing Ingest Profiles.
