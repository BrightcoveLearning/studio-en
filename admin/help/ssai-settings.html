Server-Side Ad Settings

Brightcove's Server-Side Ad Insertion (SSAI) embeds ads in your video stream to provide a TV-like viewing experience with no ad buffering.

Actions
  • Delete - Select one or more ad configurations, and click Delete to remove them from your account
  • Duplicate - Select an ad configuration, and click Duplicate to create an ad configuration copy; you can cancel, or edit and save the new configuration
View ad configurations
  • Name - The name of the ad configuration; click the name link to view the details for the ad configuration
  • Type - Either VOD or Live
  • Ad Server Response - The response format from your ad server
  • Created - The creation date for the ad configuration
Create ad configuration
  1. Click Create Ad Configuration to expand the dropdown menu.
  2. Choose one:
    • VOD - Video on Demand
    • Live - Live stream
  3. Name - The name of the ad configuration
  4. Ad Server Response - The response format from your ad server
    • DFP Ad Rules - Proprietary format for DFP/Google for Publishers
    • DFP VMAP - Video Multiple Ad Playlist
    • Smart XML - Proprietary format for FreeWheel
    • VAST 3.0 - Digital Video Ad Serving Template
  5. Ad Tag - The server URL for your ad tag
  6. Disable server-side beacons - These beacons emit events from the server to measure ad impressions and report ad errors; if disabled, events will not be fired from the server, but will be included in the ad response
  7. Click Save.
Edit ad configuration
  1. Click the ad configuration name link.
  2. Edit the ad configuration values.
  3. Click Save.
