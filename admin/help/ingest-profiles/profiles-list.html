Ingest Profiles

Ingest profiles define the number and type of renditions created for each video upload.

Show Display Name / Show API Name
Click to toggle between a list of profile display names, and profile API names.
View fields
  • Profile Name and Description - Click the name link to edit the profile
  • Profile Version (Based on your account configuration):
    • Dynamic Delivery - Stores a small set of renditions and performs real-time packaging
    • Context-Aware - Uses machine learning to optimize the number and quality of renditions needed
  • Type:
    • Standard VOD - Default VOD profile provided by Brightcove. This cannot be edited or deleted
    • Custom VOD - Custom VOD profile that you have created
    • Standard Live - Default Live profile provided by Brightcove. This cannot be edited or deleted
    • Custom Live - Custom Live profile that you have created
  • Renditions - Number of renditions that may be created
  • Display - Toggle to hide/display the profile from the Upload page
Add profile
  1. Expand New Profile.
  2. Select one:
    • Dynamic Delivery - Create a new Dynamic Delivery profile
    • Live - Create a new Live profile
Delete profile
  1. Click to delete the associated ingest profile.
  2. To confirm, click Delete Profile.
