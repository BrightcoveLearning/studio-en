Ingest Profile Renditions

Here are the video and audio renditions that can be included in your ingest profiles.

Video renditions
  • Name - Rendition name
  • Type - Rendition type; values: default, custom
  • Codec - Video compression standard
  • Bitrate - Amount of video data bits transferred within second, measured in kilobits per second (kbps)
  • Height - Video height, measured in pixels
  • Aspect Mode
    • Preserve - Preserve the aspect ratio of the video
    • Custom - Select the aspect ratio for the video
  • Profile - Type of rendition
Audio renditions
  • Name - Rendition name
  • Type - Rendition type; values: default, custom
  • Codec - Audio compression standard
  • Bitrate - Amount of video data bits transferred within second, measured in bits per second (bps)
  • Channel - Number of channels supported
