Simple Editor

Use the simple editor to create and customize your ingest profiles.

Actions
  • Use as default for Uploading - Set this profile as the default ingest profile for uploading videos
  • Use as default for Social clipping - Set this profile as the default ingest profile for social clipping
  • Duplicate - To create a similar profile, you can duplicate this profile and edit changes needed
Profile data
  • Name - Your ingest profile name
  • Description - Some descriptive text about your ingest profile
  • Store master files - Keep a copy of the master file for future retranscoding
  • Images - Specify the dimensions for the Poster and Thumbnail images
  • Capture images automatically - During upload, Video Cloud automatically captures the Poster and Thumbnail images from the 50% point of your video
  • Don't capture images - Video Cloud will not capture images. You can do this later in the Pages module.
Renditions
  • Add Rendition button - Add a video or audio rendition to your ingest profile; view the View Rendition List link to see available renditions
  • View Rendition List link - Click this link to view the available video and audio renditions
  • Codec - Video compression standard
  • Name - Rendition name
  • Type - Rendition type; values: default, custom
  • Bitrate - Amount of video data bits transferred within second, measured in kilobits per second (kbps)
  • Height - Video height, measured in pixels
