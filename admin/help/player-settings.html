Player Settings

Player settings define the type of authentication used with the Brightcove API's.

Basic Authentication

    • Enable - Check the box to enable your players to use basic username and password authentication to access the Player Management API and Delivery System API
    • Disable - Uncheck the box to disable basic authentication; in this case, API authentication must be used
  2. Click Save.
