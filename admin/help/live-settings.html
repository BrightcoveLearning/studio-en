Live Settings

Manage fill slates and credentials for Brightcove Live streams. SSAI ad configurations are managed in Server-Side Ad Settings.

Add fill slate

Fill slates are content to be served when there are gaps in a live stream ad break.

  1. Click Add Fill Slate.
  2. Enter Name - a descriptive name for your fill slate.
  3. Enter Source URL - full address of the fill slate.
  4. Click Save.
Delete Fill Slate
  1. Click to delete the associated fill slate.
  2. In the Delete dialog, click Delete to confirm.
Add credential

Live credentials are required when using either the Live module or the Live API.

  1. For Live Video Cloud credentials, start with API Authentication credentials .
    For S3, FTP or RTMP credentials, skip this step.
  2. Enter Label - an internal name for this credential
  3. Select a Type:
    • FTP - Used to save the VOD asset to an ftp location (API use)
    • RTMP - Used to create RTMP credentials to be used by the encoder
    • S3 - Used to save the VOD asset to an Amazon S3 account (API use)
    • Video Cloud - Used to save the VOD asset to a Video Cloud account (API use)
  4. Enter Public Key / Username:
    • FTP - Enter the username for the FTP site
    • RTMP - Enter the username that the encoder will use to authenticate
    • S3 - Enter the username for the S3 account
    • Video Cloud - Enter the Client ID generated for the API Authentication credential
  5. Enter Private Key / Password:
    • FTP - Enter the password for the FTP site
    • RTMP - Enter the password that the encoder will use to authenticate
    • S3 - Enter the password for the S3 account
    • Video Cloud - Enter the Client secret generated for the API Authentication credential
  6. Click Save.
Delete credential
  1. Click to delete the associated credential.
  2. In the Delete dialog, click Delete to confirm.
