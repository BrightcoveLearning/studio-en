Language Settings

Define the languages associated with your video metadata.

Note: If you delete a language that is in use, all metadata associated with the language will be hidden. To restore the associated metadata, add the language again.
Primary language

The Primary Language is initially displayed when you edit video properties in the Media module. Existing metadata will be associated with the Primary Language.

Edit primary language

  1. Click on the primary language displayed to change it.
  2. Start typing your language to see a list of values.
  3. Select a language.
  4. Click Save.
Additional languages

After a Primary Language is saved, additional metadata languages can be added. This allows you to add multi-lingual metadata in the Media module.

Add language

  1. Click Add Language.
  2. In the Language input field, Start typing your language to see values.
  3. Select a language.
  4. Click Save Language.

Edit language

  1. Click on a language in the list to edit.
  2. In the Language input field, Start typing your language to see values.
  3. Select a language.
  4. Click Save , Cancel, or .
