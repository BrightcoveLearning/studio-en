Account information

View and edit the Brightcove Beacon account information.

View account info

  • Account Name - A descriptive name for the account
  • Account ID - Assigned account ID
  • Address - Account mailing address
  • Phone - Contact phone number
  • Max Users - Maximum number of users allowed for the account. Manage users in the User Management section
Edit account info
  1. Click Edit Account Information.
  2. Update field values.
    Notes:
    • The Account ID cannot be changed
    • Max Users is based on your account type
  3. Click Save.
