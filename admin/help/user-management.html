User Management

View, add, edit, or delete users for the account.

View users list
  • Name - User name and email address; click a user name to edit module access
  • Role:
    • Standard - User does NOT have access to Admin sections
    • Administrator - User has access to Admin sections

    • Grant user access to one of more of the following modules:

    • Media - Allow users to edit videos, create playlists and publish your videos
    • Upload - Allow users to add videos to Beacon Studio
    • Analytics - Allow users to view video and player traffic data
    • Live - Allow users to broadcast live events
    • Gallery - Allow users to use templates to create video content pages
    • Audience - Allow users to connect Video Cloud analytics to marketing automation platforms, such as Oracle Eloqua, Marketo, Salesforce or HubSpot
    • Social - Allow users to publish videos to social media platforms, like YouTube, Facebook and Twitter
    • Pages - Allow users to create and manage content that is displayed in your Brightcove Beacon app
    • Monetization - Allow users to create and manage subscription plans for your Brightcove Beacon app
    • Cloud Playout - Allow users to program and schedule live events using VOD content
    • Last Login - Date and time when the user last logged in
Add user
  1. Click + Add user.
  2. Enter values for a new user:
    • Email
    • First Name
    • Last Name
  3. Role:
    • Standard - User does NOT have access to Admin sections
    • Administrator - User has access to Admin sections
  4. Module Access - Select the module(s) that you want the user to have access to.
  5. Click Save.
Edit user
  1. Select a user row to edit.
  2. You can only change the Module Access for a user.
  3. Click Save.
Delete user
  1. Click associated with a user.
  2. To confirm, click OK.
Need more users?

Each account is given a set number of users. To add more, contact customer support .
