Overview

This document details the properties used by the Server-Side connection for the viewing data integration.

Server-Side Properties


Integration Property Description
accountId Brightcove Account ID.
CustomerId Value for the unique viewer ID. (e.g., Eloqua viewer ID, Hubspot viewer ID, custom viewer ID, etc.)
videoId Brightcove Video ID.
videoName Brightcove Video Name.
VideoDuration Total length of the video in seconds.
player Brightcove Player ID.
ViewDate Date when viewing activity occurred.
MaxPercentagewatched Highest completion percent for this video by this user on a particular day.
TotalWatchDuration User's total watch time for this video on this day in Seconds.
WatchCount Number of times a user started this video on this day.
If a user replays or rewinds the video, the viewing time is added to the total watch duration. However, it is not recorded as a new watch count.
