Overview

The integration between Brightcove and Eloqua allows video engagement data to be synced as custom objects in Eloqua. To ensure that this data is correctly transferred and recorded, it’s important to validate the data directly within Eloqua’s interface.

Accessing Custom Objects

  1. Log in to your Eloqua account.

  2. From the top menu, select Custom Objects.

  3. In the Custom Objects view, Look for an object named BrightcoveViewEvent.

    Note: This object should appear after data is synchronized from Brightcove to Eloqua.

  4. Click the down arrow next to BrightcoveViewEvent.

  5. Select Custom Objects Records Data.

  6. Validate that data is coming into Eloqua.

Troubleshooting

  • Missing BrightcoveViewEvent Object:

    • Check if the data sync is enabled in the Brightcove Integration settings.

    • Confirm that the API credentials for Eloqua are correct and active.

  • No Data in Custom Object Records:

    • Wait up to 48 hours for data synchronization to complete.
