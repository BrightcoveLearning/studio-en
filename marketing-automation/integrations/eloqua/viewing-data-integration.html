Overview

Setting up the Eloqua integration using the Viewing Data Integration section exports additional granular (event-based) data. This viewing data is bound to a custom object and appended, not overwritten, on each contact sync.

For more information on the Contact Integration setup, see Contact Integration: Eloqua.

Requirements

  • Have viewer tracking enabled on your Video Cloud account.
  • Being a user with an admin role in Audience Insights.
  • Have an Eloqua account and user with access to install applications.
  • Install one of the two possible applications in your Eloqua instance.

Installing the application on your Eloqua Instance

To Install the Brightcove Publish Integration to your Eloqua account, Click the appropriate link and follow the onscreen instructions.

Brightcove Publish Integration for Eloqua (Audience Insights)

Brightcove Publish Integration for Eloqua (Videocloud)

Both applications are identical, with the exception of where you will be redirected after installing the application and granting authorization.

Adding the integration

  1. Log in to Video Cloud Studio.

  2. Open the Integrations module.

  3. Click Customer Data Platforms

  4. On the Viewing Data Integration section, click Add Integration

  5. Select Eloqua from the Destination dropdown.

  6. Select Customer Id or Viewer token from the Viewer ID property dropdown, and then click Save

  7. Login to your Eloqua account.

  8. Accept the Login to your Eloqua account.
