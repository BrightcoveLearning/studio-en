Publishing Videos to an In-Page Experience

{{ page.description }}

In-page experiences are created using the Gallery module and allow one or more videos, or a playlist, to be published on a page. The videos can appear in carousels, grids, and playlists that dynamically change before, during and after video play. Interactive elements can also be added, both inside and outside the video player. These in-page experiences can then be easily embedded in websites, landing pages, blogs, and email campaigns.

Note: You must have access to the Gallery module to be able to publish in-page experiences.
Note: For an overview of working with in-page experiences, see Overview: Creating, Editing and Publishing In-Page Experiences.

The Media module allows you to select one or more videos, or a playlist, and then will create an in-page experience that can be customized and published using the Gallery module.

Follow these steps to create an in-page experience using the Media module.

  1. Open the Media module.
  2. Select the video(s) or playlist to use in the experience. To select video(s), click the checkbox next to the video(s).

    To select a playlist, click All Playlists in the left navigation and click the checkbox next to the playlist.

  3. Click Publish and Embed > Gallery In-Page Experience.
  4. Select a template to use for the experience. In this example, the Grid template is selected.

    For information on the in-page templates, and to view some examples, see Overview of Gallery In-Page Experience Templates.

    Note: Once a template is selected, it can't be changed.
    Note: Some templates may not be available when a playlist or multiple videos are selected.
  5. Click Next.
  6. Enter an Experience Title, Description and choose a Layout Size. These values can be modified later. For information on these settings and how they are used, see Configuring the Details for an In-Page Experience.
  7. Click Create. The In-Page Experience editor will open in a new browser tab.
