Published

This section displays a list of videos scheduled and published using Brightcove Social.

With Auto Sync, you may see multiple entries for the same video:

  • One for initial sync
  • Additional for video metadata syncs

Click the video name link to see video details in the Media module.

Click the social media link to see the social platform page.

List data

For each published video, you will see the following data:

  • Status
  • Time of the event
  • Video thumbnail
  • Video name and short description
  • Link to social media account and site where the video was published
Publish videos
  1. Click Choose Videos to Publish
  2. This takes you to the Videos section.
  3. From there, you can select and publish videos.
Filter by status

Filter the list by status:

  • All Status Types - Display all status types
  • Deleted - Display videos that have been published and deleted from a social platform
  • Error - Display posts that resulted in an error
  • Hidden - Display posts where the video is private, unlisted or expired on social media
  • Processing - Display videos that are in the process of being posted to a social platform
  • Published - Display videos that were successfully posted to a social platform
  • Scheduled - Display videos that are scheduled to be posted to a social platform
  • Synced - Display videos that were synced with Auto Sync
  • Abandoned - Display videos that were Auto Synced out to a social platform and then either
    • Deleted from Video Cloud without deleting on the social platform; Auto Sync was configured to take no action when a video is deleted
    • Untagged so that it doesnâ€™t match the Auto Sync any more without deleting on the social platform; Auto Sync was configured to take no action when video metadata changes
Filter by destination

Filter the list by social platform destination:

  • All Destinations - Display videos published to all social platforms
  • Facebook - Display videos published to Facebook
  • Twitter - Display videos published to Twitter
  • YouTube - Display videos published to YouTube
Search videos

Use the search field to find videos in your account; the following fields are searched:

  • Video ID
  • Name
  • Short description
  • Long description
  • Tags
  • Reference ID
  • Custom fields
