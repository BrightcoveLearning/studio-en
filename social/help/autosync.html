Auto Sync

Automatically synchronize videos to social media channels based upon tags associated with a video.

Before you can publish videos, you must create destinations for your social accounts. In the left navigation, click Destinations.

Manually publishing videos is independent from Auto Sync. If a video is tagged to be published with Auto Sync, it will do so even if you have manually published the video.
Configure Auto Sync
  1. Click Set up Auto Sync.
  2. Enter a Name for the sync.
  3. Select the videos to distribute
    • Videos with all these tags - Videos with ALL of the tags listed should be synced
    • Videos with any of these tags - Videos with ANY of the tags listed should be synced
    • All videos in this account - All videos in the account should be synced
  4. Unless All videos in this account was selected, enter a comma separated list of tags.
  5. Exclude videos that are
    • Inactive - Inactive videos will not be synced (default)
    • Schedule restricted - Scheduled videos will not be synced
    • Geographically restricted - Videos with geo-restrictions will not be synced
  6. Click Next.
  7. Choose a publishing destination to synchronize the videos to.
  8. To create new destinations or edit any existing ones, click Manage Destinations.
  9. Click a destination to display the destination defaults.
  10. Click Next.
Define YouTube rules

Rules can be defined to handle what to do when a video is deleted from Video Cloud, when tags change, or when video metadata changes.

Rules are based upon the social media platform.

  1. Select what happens when a video is untagged or deleted from Video Cloud.
    • Delete distributed video on YouTube - The video will be removed from YouTube
    • Make distributed video - Make the YouTube video Private or Unlisted
    • No action - Nothing happens to the YouTube video
  2. Select what happens when the video metadata is changed.
    • Update video metadata on YouTube - Changes to video metadata in Video Cloud will be synchronized to YouTube
    • No action - No action is taken
  3. Click Next.
  4. Confirm the Auto Sync settings and click Finish.

View/edit Auto Sync jobs

You can view and edit the Auto Sync jobs that you have created.

  • Auto Sync Name & ID - Click a link in this column to open the sync properties
  • Tags - Click a link in this column to open the Media module with a list of videos that has the tag
  • Destination - Click a link in this column to see the destination details
  • Delete - Delete the associated sync job

Sync properties

  • Status
    • ACTIVE - Videos will be synchronized
    • PAUSED - No videos will be synchronized
  • Show - Filter the display list
  • Settings - Edit the Sync Settings; click Update to save changes
  • Delete - Delete the sync job
