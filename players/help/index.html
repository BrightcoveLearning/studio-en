Players

The Players module allows you to create and manage video players.

Players list
  • Name - Click the player name link to edit the player details. The player ID appears below the player name.
  • Updated - Most recent updated date and time
  • Version - Player version
  • Status - Player status
    • - Indicates that changes have been made to the player but the changes have not been published
    • - Indicates that the player is activated and all changes made to the player have been published
    • - Indicates that the player is inactive
Player summary
  1. Select a player name to see the summary on the right of the players list. The information includes:
    • Player name - Click the player name link to edit the player details.
    • Created - Player creation date
    • Click Player Preview to see a preview of your player in a modal window
    • Click Embed Code & URL to copy either the Preview Player or the Published Player embed code to test your player changes in your HTML page
    • Player ID - Player unique identifier
    • Name - Player name
    • Description - Player short description
    • Status - Player status
      • Active - The player is active and ready to use
      • Inactive - The player is deactivated
    • Published State - Player published state
      • Published - The player is published and has no unpublished changes
      • You have unpublished changes - You've made changes to the player; users will not see them until you publish the player
Search Players
  1. In Search Players, start typing to search for players in your account. The following fields are searched:
    • Player name
    • Player ID
    • Short description
Create player
  1. Click + Add player.
  2. Enter the Name for your player.
  3. Enter a Short Description for your player.
  4. Click Save.
When you create a new player, it is built using the latest player version as published by Brightcove.
Duplicate player
  1. Select the player that you want to duplicate.
  2. Click Duplicate.
  3. Enter the Name and Short Description for the new player.
  4. Click Save.
Delete player
  1. Select the player/players that you want to delete.
  2. Click .
  3. In the Delete Players dialog, confirm by clicking Delete.
