Introduction

The Brightcove AI Metadata Optimizer can generate metadata based on your video's text tracks.
This feature will ensure that your metadata is effortlessly optimized to suit your content better.

Using the AI Metadata Optimizer

  1. Navigate to the media module and select your video.

  2. Add text tracks to your video if you do not have text tracks.
    If you do not have text tacks in your video, the autocaption process will start.
  3. In the Overview tab, Click Generate with AI

    This process could take up to 20 seconds.

    Metadata you can generate with this process:

    • Name
    • Short description
    • Tags
    • Long description

  4. Select and Edit your metadata
    • A. Select your preferred metadata from the available options.
    • B. You can manually edit your metadata as needed.

    You can generate up to 5 sets of suggestions per session.

  5. Click on the checkmark button to select the metadata in the textbox. Notice this change is not yet in production.

    Only the selected metadata will be saved to your video.
  6. Click Save

FAQs

  • What are the limitations of this feature?
    - Metadata is created only from text tracks; visual content is not analyzed.
    - If a video lacks captions, you can choose to generate them. If so, re-transcoding with current entitlements is required.
    - If a video has multiple language transcripts or audio tracks, the tool defaults to the English track or the most recently uploaded.
  • What languages are supported for this tool?
    English (en), German (de), Spanish (es), French (fr), Hindi (hi), Italian (it), Japanese (ja), Korean (ko), Polish (pl), Portuguese (pt), Russian (ru), Turkish (tr), Chinese Simplified (zh).
  • Is there a way to approve the content so it does not get automatically "pushed" to the public domain if it is tagged incorrectly?
    Yes, all generated metadata requires your approval before it updates production values.
  • Can the Universal Translator be used to generate the text track for the metadata Optimizer?
    Yes, Metadata Optimizer can work with Universal Translator, Brightcove Autocaptions, or text tracks generated by third-party tools as long as they are in a standard captions format.
  • Will this tool work if I use captions generated from a tool outside Brightcove?
    Yes, it supports captions generated by tools outside Brightcove.
