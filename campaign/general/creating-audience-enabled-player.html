Creating an Audience-enabled Player

In this topic you will learn how to export lead form data collected through Gallery experiences.

Video engagement data will only be recorded when video content is viewed inside of an Audience-enabled Brightcove Player. There are two ways to create an Audience-enabled player:

  • Create a new player using the Audience module
  • Enable Audience on an existing player

Creating a new player using the Audience module

Before creating an Audience-enabled player using the Audience module, the Audience module should be configured with at least one data connection:

  • Eloqua
  • Marketo (Munchkin)
  • Marketo (REST APIs)
  • Salesforce
  • HubSpot (REST APIs)
  • HubSpot (Client-Side APIs)
  • SSO
  • Custom integration

Note that Audience Tracking is provided by default. To create an Audience-enabled player using the Audience module, follow these steps.

  1. Open the Audience module.
  2. Click Audience Players in the left navigation.
  3. Click Create Player. A new Audience-enabled player will be created and appear in the list of players.
  4. Using the Connect To dropdown, select the data connection that the player will use.
  5. (Optional) Using the Lead Form dropdown, select the lead form that the player will use.

Enabling Audience on an existing player

To Audience-enable an existing player, follow these steps.

  1. Open the Players module.
  2. Click on a player name to open the player properties.
  3. Click Audience in the left navigation menu.
  4. Check the Enable Audience option.
  5. Use the Connect To dropdown to select an Audience connection.
  6. (Optional) Use the Lead Form dropdown to select a lead form to use.
  7. Click Save. A message will appear indicating that you have unpublished changes.
  8. Click Publish & Embed and then Publish Changes to publish the change to the player.
  9. Return to the Audience module. Click Audience Players in the left navigation.
  10. Confirm that the player appears in the list of Audience players.
